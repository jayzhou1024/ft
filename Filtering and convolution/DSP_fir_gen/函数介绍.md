```c

DSP_fir_gen(
       const float* x,      //输入数组
	   const float* h,  //系数数组
  	   float* r,	    //输出数组
 	   int nh,	    //系数数组长度
	   int nr	    //输出数组长度
	)
```
**函数介绍**：Q15格式的fir滤波器，
**计算**：r(n) = h(0)*x(n) + h(1)*x(n−1) + ... + h(nh−1)*x(n−nh+1)   where n = {0, 1, ... , nr−1}
**说明**：Ti中初始化x时前（nh-1）为0，表示计算公式中x的负数索引

**参数要求**：

 * nh、nr无特殊要求
