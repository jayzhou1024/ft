```c
void DSP_fir_cplx (
    vector signed short *x,   
    signed short *h,       
    vector signed short  *r, /*额外开了2*r的内存空间  减少在函数内部分配内存和初始化*/
    int nh,
    int nr
)
```
**函数介绍**：Q15格式的复数fir滤波器
**计算**：r(n) = h(0)*x(n) + h(1)*x(n−1) + ... + h(nh−1)*x(n−nh+1)   where n = {0, 1, ... , nr−1}

**参数要求**：  

 *  nr >= 8; nr % 4 == 0 
 * nh >= 4; nh % 2 == 0 
 * r 16字对齐
