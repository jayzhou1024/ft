```c
void DSP_fir_r8_h16 (
    vector signed short *x,   
    signed short *h,       
    vector signed short  *r,
    int nr
)
```
**函数介绍**：Q15格式的fir滤波器
**计算**：r(n) = h(0)*x(n) + h(1)*x(n−1) + ... + h(nh−1)*x(n−nh+1)   where n = {0, 1, ... , nr−1}

**参数要求**：  

 *  nr >= 4; nr % 4 == 0 
 * nh =16
 * r 16字对齐
